Python Server Creator

Published Python

Designed a lightweight and efficient Python server capable of handling real-time requests with minimal overhead. The server provides clean endpoints, JSON responses, and support for rapid prototyping of backend systems. Ideal for APIs, automation tools, and local development environments.

server.py
1 from http.server import BaseHTTPRequestHandler, HTTPServer
2 class SimpleServer(BaseHTTPRequestHandler):
3   def do_GET(self):
4     self.send_response(200)
5     self.send_header("Content-type", "application/json")

7     self.wfile.write(b'{"message": "Python Server Running"}')
8 server = HTTPServer(("0.0.0.0", 8080), SimpleServer)
9 server.serve_forever()

Facebook Auto Share

Published Python

Developed an automated content distribution system using the Facebook Graph API. The tool allows scheduled and real-time posts to be shared directly to Facebook Pages. It supports text, images, and link previews with secure token-based authentication. Ideal for creators, brands, and automated content workflows.

fb_auto_post.py
1 import requests
2 PAGE_ID = "YOUR_PAGE_ID"
3 ACCESS_TOKEN = "YOUR_ACCESS_TOKEN"
4 def auto_share(message):
5   url = f"https://graph.facebook.com/{PAGE_ID}/feed"
6   data = {"message": message, "access_token": ACCESS_TOKEN}
7   return requests.post(url, data=data).json()
8 # Example Run
9 auto_share("Hello Facebook!")
